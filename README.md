# Even/Odd Checker

A lightweight and responsive web application built with HTML, CSS, and JavaScript that checks whether a given number is Even or Odd.

## Features
- Instant results with a clean user interface.
- Simple input validation to handle empty entries.
- Responsive design for mobile and desktop screens.

## How to use
1. Open the application in your browser.
2. Enter any number in the input box.
3. Click the "Check" button.
4. The result will be displayed immediately below.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run
Simply clone this repository or download the files and open 
[https://muhammadzaid-kh.github.io/Even-Odd-Checker/]
`index.html` in any web browser.
